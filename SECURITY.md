# Security Policy

## Supported Versions

The application is currently distributed exclusively via GitHub releases. Only the latest release and the current state of the main branch receive active updates and security fixes.

| Version | Supported |
| ------- | --------- |
| 2.0.x | :white_check_mark: |
| 1.x.x | :x: |

## Reporting a Vulnerability

If you discover any security-related issues, please do not open a public issue tracker. Instead, report it directly to the maintainer via email: nintv00@gmail.com.

What to expect:
* **Acknowledgment:** You will receive a response within 48 hours confirming the receipt of your report.
* **Updates:** We will keep you informed about the progress of the fix.
* **Resolution:** Once fixed, a new version will be published here on GitHub.
