# Contributing to GloomMaster

Thank you for your interest in improving GloomMaster! Since this is a non-commercial fan project, community help is highly appreciated.

## How to Contribute
1. Fork the repository and create your branch from `main`.
2. Ensure your code follows standard Kotlin coding conventions and Android best practices.
3. If you are adding features that use Gloomhaven assets, make sure they align with the datasets provided by the `any2cards/worldhaven` repository.
4. Open a Pull Request with a clear description of your changes.

## Reporting Bugs
Please use the Issue Tracker to report bugs. Provide steps to reproduce the issue and specify the Android version/device used.
